# Seminar Room Allocation System 🏨

## Project Description
[cite_start]An automated platform designed to manage and optimize classroom and room assignments within a seminar[cite: 7]. [cite_start]The system handles permanent schedules alongside temporary bookings, ensuring efficient space utilization while considering specific room attributes such as wing, floor, capacity, and available equipment (e.g., projectors)[cite: 8, 9, 10].

[cite_start]The system intelligently manages temporary room vacancies (e.g., during exam study breaks), ensuring these rooms are only allocated for specific dates without affecting permanent schedules[cite: 11].
